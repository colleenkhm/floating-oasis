VS Code

  • Creating a .html file

    If you create a blank .html file and just type in '!' by itself and hit enter, it will create a boilerplate html template for you, meaning that all of the head/meta tags will already be there and you only have to worry about putting things in the body.

  • Copying chunks of code

    If there is a certain piece of code or structure that you would like to duplicate on the same page, you can simply highlight and hit shift-alt-down and the same code will then be pasted right below your original code.
